Bgimg Demos

Make images become background-images for HTML elements.

Defaults

Using bgimg with the default settings.

JavaScript

Parameters set using JavaScript. 

$('.demo2').bgimg({
    backgroundImage: './assets/300x300.png',
    backgroundColor: '#D14',
    backgroundRepeat: 'repeat-x',
    backgroundSize: '50px 50px',
    backgroundPosition: 'top center'
});

Data Attributes

Parameters set using HTML5's data attributes. 

data-bgimg="./assets/200x500.png"
data-bgimg-color="rgba(72, 79, 221, 1)"
data-bgimg-repeat="no-repeat"
data-bgimg-size="contain"
data-bgimg-position="bottom right"
